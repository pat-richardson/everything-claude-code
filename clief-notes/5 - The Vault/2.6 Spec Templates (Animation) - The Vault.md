## What You'll Get From This

Ready-to-use spec templates for both long-form and short-form animated videos. Plus a prompt that generates the first draft of a spec from any script.

---

## What This Is

The spec is the most important file in the animation workflow. It is the contract between the voiceover and the animation. It maps timing, visuals, emphasis, and audio sync points beat by beat. Claude builds the animation from this file.

This download includes three things:

#### 1\. Long-form spec template (5+ minutes)

Full structure with project info, visual philosophy, beat map (timestamped), key moments, and audio sync points. Every field has a description so you know what to write.

#### 2\. Short-form spec template (under 60 seconds)

Tighter version for Reels, TikToks, and Shorts. Fewer beats, faster pacing, same principles.

#### 3\. Spec generation prompt

A ready-to-paste prompt that you give to Claude along with a script. Claude generates the first draft of a spec from it. You edit from there.

If you are going through the [**Building Animations**](https://www.skool.com/quantum-quill-lyceum-1116/classroom/d3907117?md=f7a33a9888604a08a7e48bb876682691&utm_campaign=skool_link_classroom&utm_content=9a42d1d697754c499c3effeee8135e03 "https://www.skool.com/quantum-quill-lyceum-1116/classroom/d3907117?md=f7a33a9888604a08a7e48bb876682691&utm_campaign=skool_link_classroom&utm_content=9a42d1d697754c499c3effeee8135e03") course (Level 2), these templates are what you use at the spec stage.

## One important note from experience:

Do not put code, frame numbers, or pixel positions in the spec. Keep it in plain English. Giving Claude creative room within clear boundaries produces better results than controlling every detail. The templates are designed this way on purpose.
