**What You'll Get From This**

A reference document covering Claude's built-in skills, how they work, and when to use them.

---

**What This Is**

This manual covers what skills are, how Claude reads and uses them, and how to wire them into your workspace. Think of it as a field guide for the tools layer (Layer 3) of the folder architecture.

You do not need to read this front to back. Download it, give it to Claude, and ask it to explain any section. Or ask Claude to help you find the right skill for a task you are working on.
