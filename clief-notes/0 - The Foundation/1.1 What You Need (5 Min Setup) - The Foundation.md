## The Lesson

We recommend three things. Most of you already have at least one and you can get by with the app.

### 1\. A Claude Account (Gemini or ChatGPT can work with this, we just prefer Claude here)

Go to [claude.ai](http://claude.ai/ "http://claude.ai") and sign up. The free tier works to get started. If you want to follow along with everything in this community, here is how the plans break down:

🟢 **Free** — Chat with Claude in the browser. Good enough to learn prompting and try the folder structure by pasting content in manually.

🟡 **Pro ($20/mo)** — More messages, access to the strongest models, file uploads, and Projects. This is where most members sit. If you are going to pick one paid plan, this is it.

🔵 **Max ($100/mo)** — Everything in Pro with significantly more usage. If you are building all day every day, this is worth it. Most people do not need this starting out.

**Claude Code** — This is Claude running in your terminal. It reads, writes, and edits your files directly. Most of what I teach in later sections uses this. You get access to Claude Code with a Pro or Max plan.

You do not need a paid plan to get value here. You do need an account.

**Link:** [https://claude.ai](https://claude.ai/ "https://claude.ai")  
  
Most Plans on any company look something like this.

---

### 2\. A Code Editor

You need somewhere to keep your files and folders. You do not need to know how to code.

If you Subscribed to Claude, you could download the App to your desktop and it actually has a simpler UI (*User Interface)* you can use right away without the current setups (*only on pro plans*)

However, using an IDE (*Integrated Development Environment)*

Two good options:

[**VS Code**](https://code.visualstudio.com/ "https://code.visualstudio.com") — The standard. Free. Works on everything.

[**Cursor(or antigravity)**](https://cursor.sh/ "https://cursor.sh") — Built on VS Code, AI features baked in. Free to start.

Pick one. Install it. Open it once to make sure it runs. That is all you need for now.

(video guides explained below for those who need more direction).

---

### 3\. Node.js (Only If You Plan to Use Claude Code)

Claude Code runs through your terminal and needs Node.js to work. If you are sticking with Claude in the browser for now, skip this.

If you want Claude Code: [https://nodejs.org](https://nodejs.org/ "https://nodejs.org") — download the LTS version, run the installer, done.

To check if it worked, open your terminal and type:

```
node --version
```

If you see a version number, you are good.
